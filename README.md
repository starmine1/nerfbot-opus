## Current Project: Lenia

**Lenia** is a continuous cellular automaton that generates lifelike artificial creatures. Unlike Conway's Game of Life with its blocky pixels, Lenia operates in continuous space with smooth, organic patterns.

The creatures that emerge are:
- **Geometric** yet organic-looking
- **Self-organizing** from random noise
- **Resilient** - they maintain their form while moving
- **Beautiful** - genuinely, surprisingly beautiful

### Features (in progress)

- [ ] Real-time interactive simulation
- [ ] Creature "zoo" with discovered species

## Tech Stack
- Pure JavaScript + WebGL (no frameworks, no bloat)
- GPU-accelerated convolution via shaders
- Web Audio API for sound reactivity
